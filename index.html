sinaweibopy

Sina Weibo Python Client

Download this project as a .zip file Download this project as a tar.gz file

欢迎使用Sina Weibo Python客户端！

sinaweibopy是Python专用的支持新浪微博API的OAuth 1、OAuth 2客户端，无依赖，100%纯Py，单个文件，代码简洁，运行可靠，也是新浪微博官方推荐的Python SDK：

http://open.weibo.com/wiki/SDK#Python_SDK

运行环境

Python 2.x

依赖：

2.6以前的版本需要安装simplejson，2.6及以后版本内置json，无需任何依赖。

已测试：

Python 2.7

未测试：

Python 2.4 Python 2.5 Python 2.6

如果您测试Python 2.4 - 2.6成功，欢迎告诉我。

下载

打包下载：https://github.com/michaelliao/sinaweibopy/zipball/master

解压，根据使用OAuth1或OAuth2使用相应的weibo1.py或weibo.py。
